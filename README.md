## 安装说明

**Visual Studio Code**：

「查看」->「命令面板」->「输入`代码片段`」->「代码片段：配置用户代码片段」->「输入或选择`php`/`html`」->「将本项目对应的 JSON 文本粘贴后保存」

**Sublime Text**：

打开 `Preferences > Browse Packages`，在打开的文件夹（一般为`C:\Users\用户名\AppData\Roaming\Sublime Text 3\Packages\Packages`）中解压下载的文件目录（`zblogcn`）。
